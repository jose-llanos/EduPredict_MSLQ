📖 Descripción General

La API Edu Predict MSLQ es una API REST profesional construida con FastAPI que proporciona capacidades de predicción de desempeño académico basadas en el cuestionario MSLQ (Motivated Strategies for Learning Questionnaire).

Información Técnica

Aspecto Valor
Framework FastAPI 0.104.1
Servidor ASGI Uvicorn 0.24.0
Algoritmo ML Random Forest Classifier (scikit-learn 1.3.2)
Validación de Datos Pydantic 2.5.0
URL Base http://localhost:8000
Versión 1.0.0
📌 Nota: La API incluye documentación interactiva automática en /docs (Swagger UI) y /redoc (ReDoc).

✨ Características Principales

🤖 Predicción Inteligente

Modelos Random Forest entrenados para predecir desempeño en semanas 4, 8 y 12

📊 Análisis Detallado

Proporciona predicción, nivel de desempeño y puntuación de confianza

⚡ Procesamiento en Lote

Realiza predicciones para múltiples estudiantes simultáneamente

📚 Documentación Completa

Documentación interactiva integrada con Swagger y ReDoc

✅ Validación Automática

🔒 CORS Habilitado

Soporte para solicitudes desde múltiples orígenes

🚀 Inicio Rápido

Instalación y Ejecución

cd EduPredict_MSLQ
pip install -r requirements.txt
python3 main.py

La API estará disponible en http://localhost:8000

Verificar Que Funciona

curl -X GET "http://localhost:8000/api/v1/health" \
  -H "accept: application/json"
💡 Tip: Usa Swagger UI en http://localhost:8000/docs para una experiencia interactiva más cómoda.

🔌 Endpoints

1. Health Check

GET /api/v1/health

Verifica el estado de la API

2. Predicción Individual

POST /api/v1/predict

Realiza predicción individual de desempeño académico

Parámetros de Request:

week integer REQUERIDO

Semana de predicción (4, 8 o 12)

practica_1 number

Calificación de práctica 1 (0.0 - 5.0)

practica_2 number OPCIONAL

Calificación de práctica 2 (requerida para semanas 8 y 12)

practica_3 number OPCIONAL

Calificación de práctica 3 (requerida para semana 12)

I11-I75 number REQUERIDO

Características del cuestionario MSLQ (14-17 características según la semana)

3. Predicción de Estudiante

POST /api/v1/predict-student

Predicción identificada por nombre de estudiante

4. Predicción en Lote

POST /api/v1/predict-batch

Predicciones para múltiples estudiantes

5. Información de la API

GET /api/v1/info

Información sobre modelos y características disponibles

📦 Modelos de Datos

PredictionResponse

Campo Tipo Descripción
prediction integer Valor predicho (0=Bajo, 1=Medio, 2=Alto)
performance_level string Descripción del nivel de desempeño
performance_range string Rango de calificación (ej: "3.0 - 4.0")
week integer Semana pronosticada
confidence_score number Puntuación de confianza (0.0 - 1.0)

Niveles de Desempeño

Código Nivel Rango
0 Bajo desempeño 0.0 - 2.9
1 Medio desempeño 3.0 - 4.0
2 Alto desempeño 4.1 - 5.0

📦 Instalación Completa

Requisitos

  • Python 3.9+
  • pip (gestor de paquetes)
  • Archivos CSV de datos en la carpeta data/

Pasos

# 1. Navegar a la carpeta del proyecto
cd EduPredict_MSLQ

# 2. Instalar dependencias
pip install -r requirements.txt

# 3. Copiar datos (si no están presentes)
cp ../data-model-mslq-pretest-207.csv data/

# 4. Ejecutar la API

# 5. Abrir en navegador
# Swagger UI: http://localhost:8000/docs
# ReDoc: http://localhost:8000/redoc
